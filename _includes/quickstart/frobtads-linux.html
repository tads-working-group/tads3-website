1. Open a terminal with $CTRL + ALT + T.
  2. Check to verify the following packages are installed. This can be done with $which package-name (Ubuntu/Debian) or $rpm -ql package-name (Fedora/CentOS).
    1. cmake
    2. g++
    3. make
  3. If you are missing any of these three packages (indicated by a blank response), then install them. This can be done with $sudo apt install cmake g++ make (Ubuntu/Debian) or $sudo dnf install cmake gcc-c++ make (Fedora/CentOS).
  4. Verify/install **ncurses** with $sudo apt-get install libncurses5-dev libncursesw5-dev (Ubuntu/Debian) or $sudo dnf install ncurses-devel (Fedora/CentOS).
  5. Download and extract a copy of the latest FrobTADS source code.
    (It is recommended to move the extracted directory frobtads-2.0 to your home ~ directory.)
