TADS 3 — The Text Adventure Development System

TADS 3 is the most powerful and flexible interactive fiction authoring system ever created, designed and built with those who are already comfortable with the conciseness, specificity, and clarity of programming in mind. Step into a world of awe-inspiring simulation, dynamic runtime capabilities, and audiovisual splendour to rival Vorple!

TADS 3 comprises an object-oriented programming language, the TADS compiler, a virtual machine called T3 for playing compiled games, and adv3, a rich library for developing IF and building a game world.
