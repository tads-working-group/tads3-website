1. Open a terminal with CTRL + ALT + T.
  2. Check to verify cmake is installed. This can be done with $which cmake (Ubuntu/Debian) or $rpm -ql cmake (Fedora/CentOS).
  3. If you are missing this package (indicated by a blank response), then you are likely missing the required dependencies as well. These can be installed with $sudo apt install cmake g++ make (Ubuntu/Debian) or $sudo dnf install cmake gcc-c++ make (Fedora/CentOS).
  4. Verify/install your ncurses package with $sudo apt-get install libncurses5-dev libncursesw5-dev (Ubuntu/Debian) or $sudo dnf install ncurses-devel (Fedora/CentOS).
  5. Download and extract a copy of the latest FrobTADS source code.
    (It is recommended to move the extracted frobtads-2.0 directory to your home ~ directory.)
  6. Navigate to the extracted directory through your terminal. For example: $cd ~/frobtads-2.0
  7. Make sure you are actually in the directory. $ls should list filenames such as:
    • CMakeLists.txt
    • CMCommon.cmake
    • etc...
    If you only see frobtads-2.0, then try $cd frobtads-2.0 && ls to repeat this verification step.
  8. $mkdir build
  9. $cd build
  10. Next, we will prepare the compiler by choosing where the resulting binaries will go. By default, the installation prefix is /usr/local, which places the binaries into /usr/local/bin/, making them available as terminal commands.
    1. Default installation path
      Simply run $cmake ..
    2. Custom installation path
      You can choose which directory the binaries will be installed to by changing the -DCMAKE_INSTALL_PREFIX=<path-prefix> option.
      Notice that this prefix is set in two places in this command:
      $CFLAGS="-DCMAKE_INSTALL_PREFIX=/usr/local" CXXFLAGS="-DCMAKE_INSTALL_PREFIX=/usr/local" cmake ..
  11. Finally, run the compile process with this command:
    $cmake --build . --target install
  12. Assuming the compile process is successful, you will have two new binaries, which will become particularly useful:
    • $frob MyNewGame.t3 for playing TADS game files in the terminal, and
    • $t3make in a project directory for compiling TADS 3 games.
    At this point, we are not done, but we have successfully finished a major series of steps!
